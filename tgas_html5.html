TAG's Semânticas

TAG's Semânticas são tags que o buscador do google consegui entender e é importante pra funcionar melhor no algoritimo de busca do google.
Algumas TAG's semanticas foram incluidas no novo HTML5 como a TAG header (cabeçalho); antes de ter essa tag era comum usar a tag DIV (inclusive é só assim que eu sei).
Vamos as TAG's:

TAG - HEARD - essa tag funciona exatamente igual a tag div, mas é semântica;
TAG - NAV - essa tag é melhor pro google entender que é uma navegação, ou seja, ao invês de usar uma div dentro de outra pra criar um cabeçalho é melhor substituir a segunda div por nav;
